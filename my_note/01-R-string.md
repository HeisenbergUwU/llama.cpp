## 实验 1: R"()" 原始字符串

### 日期
2026-04-20

### 目标
理解 C++11 引入的原始字符串字面量 `R"()"`

### 代码
```cpp
#include <cstdio>

const char * LLAMA_ASCII_LOGO = R"(
▄▄ ▄▄
██ ██
██ ██  ▀▀█▄ ███▄███▄  ▀▀█▄    ▄████ ████▄ ████▄
██ ██ ▄█▀██ ██ ██ ██ ▄█▀██    ██    ██ ██ ██ ██
██ ██ ▀█▄██ ██ ██ ██ ▀█▄██ ██ ▀████ ████▀ ████▀
                                    ██    ██
                                    ▀▀    ▀▀
)";

int main() {
    printf("%s\n", LLAMA_ASCII_LOGO);
    return 0;
}
```

### 编译命令
```bash
# 成功
clang++ LOGO.cpp -std=c++23 -o LOGO && ./LOGO

# 失败（链接错误）
g++ LOGO.cpp -std=c++11 -o LOGO  # 需要 -lc++ 或改用 printf
```

### 关键点
1. **`R"()"` 语法**: C++11 引入，用于原始字符串
2. **优点**: 
   - 不需要转义字符（如 `\`, `"`, `\n`）
   - 支持多行字符串
3. **编译器要求**: 需要 C++11 或更高版本
4. **macOS 支持**: 完全支持，使用 `clang++` 或 `g++ -std=c++11`

### 遇到的问题
1. **链接错误**: 使用 `std::cout` 时需要链接 C++ 标准库
   - 解决：改用 `printf` 或使用 `clang++`
2. **Unicode 字符**: 某些编译器可能不支持 UTF-8 特殊字符
   - 解决：使用 `clang++` 或简化为 ASCII 字符
